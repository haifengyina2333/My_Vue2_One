# 项目说明

## 项目相关依赖安装
```
npm install
```

### 运行项目
```
npm run serve
```

### 使用开发者模式运行项目
```
npm run build
```

### 进行代码规范检查和修复
```
npm run lint
```

### 项目技术要点:

- 对axios进行二次封装,便于使用(request下的request.js)
- 整和所有api地址为一个文件,方便管理(request下的api.js)
